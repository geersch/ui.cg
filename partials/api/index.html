Quick Start

Clone the repository:

$ git clone git://github.com/geersch/ui.cg.git

Install node (http://nodejs.org/).

Run nmp install to install the dependencies:

$ npm install

Build it:

$ grunt build

When it has been built, you can find the redistributables (minified & uncompressed) in the dist/ folder.

You only need to add the module to your AngularJS module:

angular.module('myModule', ['ui.cg']);
