numberinput
directive in module cg.ui

Description

Input for a floating point number with a configurable decimal separator (default: .) and decimals (default: 2).

Usage

as element:
<numberinput
       ng-model="{string}"
       [decimal-separator="{string}"]
       [decimals="{number}"]
       [step="{number}"]
       [mousewheel="{boolean}"]
       [keyboard="{boolean}"]
       [spinner="{boolean}"]
       [maximium="{number}"]>
</numberinput>

Parameters

ParamTypeDetails
ngModelstring

Assignable angular expression to data-bind to.

decimal-separator
(optional)
string

The decimal separator to display (default: .).

decimals
(optional)
number

The maximum number of allowed decimals (default: 2).

step
(optional)
number

The value to increment or decrement with (default: 1).

mousewheel
(optional)
boolean

Whether user can scroll inside the input to increase or decrease the value (default: true).

keyboard
(optional)
boolean

Whether user can increase of decrease the value using the keyboard up/down arrows (default: true).

spinner
(optional)
boolean

Whether or not spin buttons are shown (default: true).

maximium
(optional)
number

The maximum allowed value.
